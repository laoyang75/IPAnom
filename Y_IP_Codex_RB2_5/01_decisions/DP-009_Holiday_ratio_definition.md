# DP-009：“假日占比”口径定义（当前输入仅含周末）

决策：你已选择 **选A**（周末≈假日）。

背景：你提出“假日占比”。在当前唯一输入锚点（W 源表）中，存在：
- `周末上报次数`
- `工作日上报次数`
但未发现“节假日/假日”字段。是否把“周末”近似为“假日”必须先合同固化，避免后续口径漂移。

你只需回复：`选A` / `选B` / `选C`

## 选项

### A（推荐：周末≈假日）
定义：
- `holiday_reports = 周末上报次数`
- `holiday_ratio = 周末上报次数 / NULLIF(上报次数,0)`

并在字段命名上明确来源（建议字段名用 `weekend_*`，展示层再映射为“假日”）。

### B（不提供假日占比）
本期 v2.0 不输出“假日占比”字段，仅输出 `weekend_report_ratio`（周末占比）。

### C（保留字段但置空）
输出 `holiday_ratio` 字段但固定为 NULL（占位），等待未来引入“节假日日历/节假日口径”后再补齐。

## 固化位置（被选中后）

- `02_contracts/metric_contract.*`：假日/周末的定义与边界（`上报次数=0` 时的处理）
- `02_contracts/schema_contract.*`：成员表与块级画像表字段是否包含 `holiday_*`
- `CHANGELOG.md`：记录原因与影响
